# Readme

## About

This package is a modern discord api library allowing you to interface with discord's api

## Ultimate-goal

to use and support

- the latest python features
- free-threaded python versions for performance

## API Reference

This project implements Discord's official API. For details on available endpoints, their parameters, and specifications, refer to the official Discord API documentation:

- **Official API Docs:** [discord/discord-api-docs](https://github.com/discord/discord-api-docs)
- **Project Guidelines:** See [.github/API_REFERENCE.md](.github/API_REFERENCE.md) for how we use the Discord API documentation
